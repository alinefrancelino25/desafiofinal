🎤 ChatGPT por Voz com Whisper

Este projeto implementa um sistema de conversa por voz com o ChatGPT utilizando tecnologias de Inteligência Artificial.

🚀 Tecnologias utilizadas
Python
Whisper (OpenAI)
ChatGPT API
gTTS (Google Text-to-Speech)
🧠 Como funciona
O usuário fala
O áudio é convertido em texto (Speech-to-Text)
O ChatGPT gera uma resposta
A resposta é convertida em áudio (Text-to-Speech)
📌 Objetivo

Demonstrar a integração de múltiplas tecnologias de IA para criação de assistentes virtuais.
